Panduan Penggunaan Aplikasi

1. Clone Repository

Clone repository ini menggunakan perintah:

git clone https://github.com/sylent-sys/database-metadata-extractor.git

2. Setup Lingkungan Virtual

Buat dan aktifkan lingkungan virtual:

# Untuk Windows
python -m venv venv
venv\Scripts\activate

# Untuk Linux/Mac
python3 -m venv venv
source venv/bin/activate

3. Install Dependencies

Instal semua dependensi dari requirements.txt:

pip install -r requirements.txt

4. Konfigurasi

Peringatan Keamanan: File konfigurasi asli (config.json) sudah di-ignore oleh Git. Pastikan tidak pernah mengunggah file konfigurasi dengan data asli ke repository!

Salin file contoh konfigurasi:

cp config.example.json config.json

Edit file config.json dengan editor teks:

{
  "server": "alamat_server_anda",
  "real_server": "alamat_real_server_anda",
  "username": "username_database",
  "password": "password_database",
  "databases": ["DB1", "DB2", "DB3", "DB4"],
  "tables": [
    ["Tabel1", "Tabel2"],
    ["Tabel3", "Tabel4"],
    ["Tabel5"],
    ["Tabel6", "Tabel7", "Tabel8", "Tabel9", "Tabel10"]
  ],
  "output_file_pattern": "data_{timestamp}.xlsx"
}

5. Menjalankan Aplikasi

Jalankan script utama dengan perintah:

python main.py

Hasil akan disimpan dalam format Excel di file dengan nama sesuai pola data_YYYYMMDD_HHMMSS.xlsx

6. Troubleshooting

  • Jika muncul error "Driver not found", pastikan Anda telah menginstal ODBC Driver 17 for SQL Server
  • Pastikan semua informasi koneksi di config.json benar
  • Pastikan port database tidak diblokir firewall
